Cards Against Humanity
======================

Multiplayer, real time implementation of the popular card game [Cards Against Humanity][1].

After you clone: 

      git submodule update --init --recursive
      npm install

Run on dev:
       
       npm run-script dev
       Visit http://localhost:3000/

Run on prod:

       npm run-script prod

[1]: http://cardsagainsthumanity.com/
